📌 策略说明

策略决定了机器人何时买入、何时卖出。新手建议使用默认策略，熟悉后再自定义。

📊 选择策略

选择一个已上传的策略文件

📈 RSI 指标参数

RSI 是判断超买超卖的指标。值越低越适合买入，值越高越适合卖出。
RSI 低于此值时考虑买入。30 表示超卖，适合抄底。
RSI 高于此值时考虑卖出。70 表示超买，适合止盈。

📝 订单类型

限价单可以控制价格，但可能不成交
市价单成交快，但价格可能不理想

📌 风险控制说明

止损止盈是保护资金的重要手段。强烈建议设置止损，避免单笔亏损过大。

🛡️ 止损设置

亏损达到这个比例自动卖出。如 -10 表示亏损 10% 止损。建议 -5 到 -15。
价格上涨时，止损线跟随上移，锁定更多利润。
价格回撤多少触发止损
盈利多少后开始追踪

🎯 止盈设置 (ROI)

设置不同时间段的盈利目标，达到目标自动卖出。

⏰ 订单超时

限价单多久未成交自动取消
卖出订单多久未成交自动取消
